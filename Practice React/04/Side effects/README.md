# Side Effects

Kleines React/TypeScript-Projekt zum Ueben von `useState`, Props, Event-Handlern und `useEffect`.

Die App zeigt eine Box und mehrere Buttons. Beim Klick auf einen Button wird eine Farbe im State gespeichert. `useEffect` reagiert auf diese Aenderung und setzt die passende Tailwind/DaisyUI-Farbklasse fuer die Box.
