Mein zweites BackendProjekt. Express API mit Prisma in Typescript.

Features:

- Register User
- Login
- Authentisierung des Users
- Authorisierung des Users
  -> JWT; AccessToken und RefreshToken, Refresh Rotation.
- Neon Datenbank mit Prisma initialisiert

Fokus auf sauberer Struktur, sinnvolle Auslagerung.
